# 数字人格测试

> 测一测你在数字世界里的真实模样！3 分钟快速定位你的数字人格画像。

## 功能特性

- **3 种测试模式**：快测（6 题）/ 标准（16 题）/ 深度（32 题）
- **32 道精选题目**：涵盖信息获取、社交行为、表达创作、注意力沉浸 4 个维度
- **8 种人格类型**：信息猎人、社交蝴蝶、数字达人、虚拟隐士、全能玩家、围观群众、领航探索者、内容创客
- **趣味可视化**：雷达图 + 条形图 + SVG 卡通头像
- **幽默解读报告**：轻松风格，网络梗加持
- **一键分享**：支持微信、微博、小红书

## 技术栈

- Vue 3 + Composition API
- Vite 5
- 纯前端，无需后端

## 快速开始

```bash
# 安装依赖
npm install

# 开发预览
npm run dev

# 生产构建
npm run build
```

## 算法说明

基于用户作答，计算 4 个维度的得分：

- **主动搜索强度**：信息获取的主动程度
- **社交活跃度**：互动、分享、评论的活跃程度
- **表达创作度**：内容创作与观点输出的意愿
- **沉浸体验度**：数字内容消费的深度与时长

结合搜索 × 社交两个核心维度，匹配 8 种人格类型之一。
